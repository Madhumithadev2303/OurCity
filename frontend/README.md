# Crowd Complaint - Frontend

This is the frontend application for the Crowd Complaint system, built with **Vite** and **React**.

## Prerequisites
You need **Node.js** and **npm** installed on your machine to run this project.
Download them from: [https://nodejs.org/](https://nodejs.org/)

## Setup & Run

1.  Open a terminal in this `frontend` folder.
2.  Install dependencies:
    ```bash
    npm install
    ```
3.  Start the development server:
    ```bash
    npm run dev
    ```
4.  Open your browser at `http://localhost:5173` (or the port shown in the terminal).

## Features
- **Login**: Use `demo` / `demo` to bypass backend check if the backend is offline.
- **Citizen Dashboard**: View and file complaints.
