# OurCity 🏙️

OurCity is a modern, community-driven platform that empowers citizens to easily report local infrastructure issues, upvote urgent problems, and track real-time resolutions by municipal workers on an interactive live map. 

### Core Features
- **Report & Track:** Snap a photo and pinpoint issues on a live map.
- **Community Power:** Upvote issues to signal urgency to the authorities.
- **Live Updates:** Watch as issues go from "Pending" to "Resolved" with before-and-after photo proof.

### Tech Stack
- React / Vite / Leaflet (Frontend)
- Java / Spring Boot (Backend) 

### Quick Start
1. **API:** `mvn spring-boot:run` (in `/backend`)
2. **App:** `npm run dev` (in `/frontend`)
